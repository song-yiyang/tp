![Ui](docs/images/Ui.png)

**ScamBook is a desktop application for call-based scammers to maintain and track their database of contacts.** It is optimized for tech-savvy typing-preferred users, with a greater focus on CLI features compared to GUI.

* If you are interested in using ScamBook, head over to the [User Guide](https://ay2526s2-cs2103t-t16-1.github.io/tp/UserGuide.html).
* If you are interested about developing ScamBook, the [Developer Guide](https://ay2526s2-cs2103t-t16-1.github.io/tp/DeveloperGuide.html) is a good place to start.
